# 📈 Stock Trading Platform (Java)

## 📌 Description
A console-based stock trading application developed in Java that allows users to simulate buying and selling of stocks, manage their portfolio, and track balance using object-oriented programming concepts.

## 🚀 Features
- View available stocks
- Buy stocks
- Sell stocks
- View portfolio details
- Track account balance

## 🛠 Technologies Used
- Java
- OOP (Object-Oriented Programming)
- HashMap
- Scanner

## ▶️ How to Run
1. Compile the program:
   javac *.java

2. Run the program:
   java StockTradingPlatform
